There should be 3 directories: `highway`, `office` and `pedestrian`.  
Those datasets can be downloaded from <https://www.changedetection.net/>

```text
├───highway
│   ├───groundtruth
│   ├───input
│   └───temporalROI.txt
├───office
│   ├───groundtruth
│   ├───input
│   └───temporalROI.txt
└───pedestrian
    ├───groundtruth
    ├───input
    └───temporalROI.txt
```
